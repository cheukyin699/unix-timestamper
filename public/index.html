Timestamp Microservice

Hello

This is the timestamp microservice, where you can pass a string/unix timestamp into the system, and it would output both formats in json form. If it is an invalid input (i.e. doesn't contain a timestamp or date), then it would output null for both.

Examples

Usage:
/api/December%2015,%202015
/api/1450137600

Output:
{ "unix": 1450137600, "natural": "December 15, 2015" }
